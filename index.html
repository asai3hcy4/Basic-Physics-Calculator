I started this project after receiving a rejection from SIT after applying for the Bachelor of Engineering Mechanical Engineering programme. Despite being overseas, tired, and extremely tired, I spent the following days re-learning Python from scratch and rebuilding the physics formulas I studied during my Diploma in Aerospace Engineering.

This project is intentionally simple as I want it to reflect where I am right now skill wise, but it is not where I intend to stop. The goal was to write code I actually understand, line by line.

Formulas Covered
Final Velocity v = u + at
Displacement s = ut + ½at²
Force F = ma
Kinetic Energy KE = ½mv²
Gravitational Potential Energy GPE = mgh
How to Run
  1. Make sure Python 3 is installed on your machine
  2. Download main.py from this repository
  3. Open your terminal and navigate to the folder
  4. Type python main.py and press Enter
  5. Follow the menu to pick a formula and enter your values

# example function from main.py

def final_velocity():
    print("\n-- Final Velocity --")
    print("Formula: v = u + at")
    u = float(input("Enter initial velocity (m/s): "))
    a = float(input("Enter acceleration (m/s2): "))
    t = float(input("Enter time (s): "))
    v = u + a * t
    print("Final velocity =", v, "m/s")
